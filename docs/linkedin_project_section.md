## Title
Salary vs. Cost of Living Analysis — Data Analyst (AR, USA, Remote LATAM)

## Description
Comparative analysis of Data Analyst compensation across three employment scenarios (Argentine local, Remote LATAM contractor, US on-site) and six cities, measuring real purchasing power by factoring in rent, utilities, transportation, groceries, and health insurance. Built a synthetic dataset calibrated against 10+ public sources (SysArmy, BLS, INDEC, Glassdoor, Near, Zonaprop), generated 6 publication-ready charts with Python/Matplotlib, and created an interactive React dashboard with seniority toggles and dynamic KPIs. Key finding: a Semi Senior remote worker from Corrientes achieves a 7.0x salary-to-cost ratio — the highest of all scenarios analyzed.

Tech: Python · Pandas · Matplotlib · React · Recharts · CSV
